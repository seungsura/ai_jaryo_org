# Speaker Notes Stage

이 디렉터리는 deck이 안정된 뒤 발표 대본과 전환 문장을 준비하는 단계입니다.

## 구성

- `slide-to-script-map.md`: slide와 notes 대응표
- `notes/`: section 또는 slide 단위 대본 파일

## 원칙

- speaker notes는 canonical prose를 대체하지 않습니다.
- notes는 spoken flow에 최적화하되, 새 factual claim을 추가하지 않습니다.
- slide order가 바뀌면 `slide-to-script-map.md`를 먼저 갱신합니다.
