# Upstream Minimal Light Reference Summary

이 문서는 `make-slide/themes/minimal-light/README.md`와 `reference.html`의 upstream 요약이다. Jaryo active rule이 아니다.

Jaryo 판단은 아래 active 문서에서 한다.

- `docs/03-html/shared/slide-quality-rules.md`
- `docs/03-html/shared/design.md`
- `docs/03-html/shared/minimal-light-adaptation.md`

Upstream `minimal-light`는 밝은 배경, 정교한 typography, 약한 shadow로 메시지를 전면에 두는 clean presentation theme다. Jaryo deck에는 generic English label, upstream slide type, upstream use case, upstream size scale을 그대로 적용하지 않는다.
