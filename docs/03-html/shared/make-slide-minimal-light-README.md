# Minimal Light

## Design Philosophy / 디자인 철학

**EN:** A clean, bright theme inspired by Apple's presentation style. White backgrounds, precise typography, and subtle shadows create a professional, polished look. The design disappears so your message stands out.

**KO:** Apple의 프레젠테이션 스타일에서 영감을 받은 깨끗하고 밝은 테마입니다. 화이트 배경, 정교한 타이포그래피, 은은한 그림자로 전문적이고 세련된 느낌을 줍니다. 디자인이 사라지고 메시지만 남습니다.

## Use Cases / 적합한 용도

**EN:**
- General presentations and seminars
- Corporate meetings and reports
- Academic lectures and workshops
- Product introductions
- Team updates and project reviews
- Well-lit room presentations

**KO:**
- 일반 발표 및 세미나
- 기업 회의 및 보고
- 학술 강의 및 워크숍
- 제품 소개
- 팀 업데이트 및 프로젝트 리뷰
- 밝은 환경에서의 발표

## Slide Types / 슬라이드 타입

`reference.html` includes examples of all supported slide types:

1. **Title** — Overline date + large headline + subtitle
2. **Agenda/TOC** — Numbered items with descriptions
3. **Content** — Body text with accent highlights
4. **Quote** — Large quotation mark + italic text + attribution
5. **Comparison** — Before/After columns
6. **Flow/Steps** — Horizontal process with arrows
7. **Cards** — Feature cards with icons
8. **Data/Chart** — CSS bar chart
9. **Code Block** — Syntax-highlighted code (Prism.js)
10. **Image Placeholder** — Dashed placeholder area
11. **Closing** — Thank you slide
