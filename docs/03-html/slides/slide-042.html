CHAPTER 04

RAG vs Context Hub

RAG

넓은 저장소에서 passage 검색

  • 질문마다 관련 조각을 생성 입력에 주입
  • 벡터 인덱스와 retriever 품질이 결과를 좌우
  • 내부 위키·정책·긴 참고 자료에 적합

Context Hub MCP

지정한 최신 문서를 컨텍스트로 주입

  • 공식·버전별 API 문서를 필요할 때 가져옴
  • 라이브러리 ID·버전·언어로 범위를 좁힘
  • SDK/API 변경이 잦은 작업에 적합
선택 기준

넓게 찾을 때는 RAG, 지금 맞는 문서를 확인할 때는 Context Hub

Sources: Lewis et al. 2020 · MCP Docs · Context Hub / Context7
