# Provenance Docs

이 폴더는 `docs/01-sources/maps/`보다 한 단계 뒤에 있는 배경성 provenance를 관리합니다. 세미나 본문과 직접 1:1로 연결되는 section map이 아니라, 인물, 기관, 외부 repo audit 같은 supporting context를 둡니다.

## 현재 문서

- `people-and-orgs.md`: 발표에서 반복 등장하는 인물과 기관의 배경
- `external-repo-audit.md`: 외부 curated repo 검증 기록

## 원칙

- 본문 연결 map은 `../maps/`에 둡니다.
- 이 폴더는 narrative를 늘리지 않기 위한 배경성 provenance를 담당합니다.
- `Harness`, `Prompt`, `Context`, `Spec` 같은 핵심 term은 한국어 문장 안에서도 English로 유지합니다.
