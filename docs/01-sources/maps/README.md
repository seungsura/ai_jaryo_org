## 현재 문서

- `seminar-claims-and-sources.md`
- `claude-code-mastery-playbook-to-seminar.md`
- `prompt-context-harness-to-seminar.md`

## 원칙

- map 문서는 prose를 대체하지 않습니다.
- map 문서는 source provenance와 섹션 연결을 명시하는 데 집중합니다.
- 배경 조사나 인물 설명은 `../provenance/`로 보냅니다.
