세션 격리와 상태 관리

물리적 경계와 외부 상태 저장소로 컨텍스트 드리프트 방어

METHOD 01

Isolation

물리적 격리
cmux와 Git Worktree를 사용하여 작업 세션을 독립적으로 운영하고 간섭을 차단합니다.

METHOD 02

Control

맥락 제어
/compact, /clear 명령을 통해 불필요한 기억이 잡음으로 남지 않도록 컨텍스트를 정제합니다.

METHOD 03

Issue

상태 저장소
휘발되는 채팅 대신 이슈를 작업의 목적과 기준을 담는 영구적인 상태 저장소로 활용합니다.
