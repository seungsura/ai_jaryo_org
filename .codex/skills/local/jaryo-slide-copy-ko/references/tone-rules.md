# Korean Slide Tone Rules

## Slide Copy

- 기본 언어는 한국어
- 공손체 금지
- 보고서 문장보다 발표용 clause 우선
- 한 장에 한 목적 유지
- body는 스캔 가능해야 함
- title은 목적을 직접 압축해야 함
- `theme-minimal-light`에서는 한 줄당 밀도가 더 쉽게 드러나므로 lead와 body를 더 짧게 유지
- agenda, comparison, card 계열은 1~2줄 단위 rhythm을 우선
- generic English label보다 구조적으로 필요한 Korean wording을 우선

## Speaker Notes

- 말로 읽을 수 있어야 함
- deck의 주장과 순서를 바꾸지 않음
- 장표보다 조금 더 연결적이어도 되지만, 설명이 장황해지면 안 됨
- next-slide preview와 transition 문장은 짧고 분명해야 함

## Warning Signs

If any of these appear, expect a review failure:

- 직역체라서 말맛이 없음
- 명사만 과도하게 쌓여 있음
- 한 문장이 너무 길어 slide를 읽는 속도를 깨뜨림
- prose처럼 완결된 설명문이 되어 버림
- 장표의 claim보다 body가 길고 무거움
