# Gemini Invocation Contract

Always use this fixed model for Korean slide writing:

```bash
gemini -m gemini-3.1-pro-preview
```

## Prompt Ingredients

Every invocation should include:

- slide purpose
- one-sentence claim
- supporting points
- must-keep terms
- do-not-say items
- target family
- target output form
  - slide copy
  - speaker notes

## Never Delegate To Gemini

- slide count decisions
- family assignment
- chapter ordering
- evidence policy
- HTML generation

These decisions belong to planning or HTML stages, not the Korean wording stage.
